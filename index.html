Main Projects

HFCommunity

Description: The Community database of Hugging Face Hub. HFCommunity offers an offline up-to-date relational database built from the data available at the Hugging Face Hub, thus providing queryable data about the repositories hosted in the Hub.

Website: https://som-research.github.io/HFCommunity/
